カスタムHTMLタグとは？

GTMのカスタムHTMLタグを使うと、自分で自由にHTMLやJavaScriptを埋め込み、より柔軟に計測や動作を追加できます。標準タグにない外部スクリプトの挿入や、自作トラッキング用コードなど幅広く利用可能です。

簡単なサンプルコード

<script>
  alert('カスタムHTMLタグが実行されました！');
</script>

このようにJavaScriptコードを記述し、GTMの任意のトリガーで発火させます。

もう少し実用的な例

<script>
  const div = document.createElement('div');
  div.style.position = 'fixed';
  div.style.bottom = '50px';
  div.style.right = '30px';
  div.style.padding = '10px 15px';
  div.style.background = '#4F86F7';
  div.style.color = 'white';
  div.style.borderRadius = '8px';
  div.style.boxShadow = '0 4px 12px rgba(0,0,0,0.3)';
  div.style.zIndex = '9999';
  div.textContent = 'GTMカスタムHTMLタグからのメッセージです';
  document.body.appendChild(div);
</script>

例えば、こんな感じで画面にメッセージを動的に表示する用途にも利用できます。

注意点

  • JavaScriptの記述ミスによるサイト影響に注意（コンソールエラーや表示崩れなど）
  • 外部スクリプト読み込みの場合は遅延を考慮
  • プレビューで動作確認は必須
